## Baby Monitor - Pixel-diff Empty Detection
- Added pixel-diff empty bassinet detection to skip OpenAI API calls when bassinet is empty
- Uses ffmpeg to compute center-crop pixel difference against previous frame
- Calibrated against 811 labeled frames: threshold=5, zero real baby misses
- 1 false skip in backtest was actually an OpenAI false positive (no real baby in frame)
- ~33% API call savings (~120 calls/day, ~$1-2.50/day saved)
- New JSONL fields: `detectionMethod` ("pixel-diff" or "openai-vision"), `diffScore`
- Reference empty frames saved in `data/references/`
- Added `--backtest` mode to monitor.py for testing changes against historical data
  - `--backtest --quick` tests pixel-diff gate only
  - `--backtest --count N` / `--from-date YYYY-MM-DD` for filtering
